Die drei Typen
Schwache Verben
Stamm + te
Regelmäßige Verben. Der Stamm bleibt gleich, es wird einfach -te angehängt.

Endet der Stamm auf -t oder -d → -ete (z.B. arbeiten → arbeitete).
Verben auf -ieren → -ierte (z.B. studieren → studierte).
machen → machte  |  hören → hörte
Starke Verben
Vokalwechsel
Unregelmäßige Verben. Der Stammvokal ändert sich — diese Formen müssen auswendig gelernt werden.

Häufige Muster: i → a · ei → ie · e → a · ie → o · a → u
trinken → trank  |  schreiben → schrieb
Trennbare Verben
Präfix ans Ende
Das Präfix (Vorsilbe) wandert ans Satzende. Der Verbstamm folgt den Regeln der starken/schwachen Verben.

Wichtig: Im Präteritum bleibt das Präfix am Verb — es trennt sich nicht im Hauptsatz.
anrufen → rief an  |  aufstehen → stand auf
Eselsbrücke für starke Verben — häufige Vokalwechsel:
i → a   trinken→trank, finden→fand, singen→sang
ei → ie   schreiben→schrieb, bleiben→blieb, leihen→lieh
e → a   sprechen→sprach, geben→gab, lesen→las, essen→aß
ie → o   fliegen→flog, verlieren→verlor, umziehen→zog
a → ie   schlafen→schlief, laufen→lief, fahren→fuhr
