Grammatik Übersicht

Verben mit Dativ
und Akkusativ

Verben, die zwei Objekte verlangen — eine Person im Dativ und eine Sache im Akkusativ.

Die Regel Satzmuster Verbliste
Dativobjekt
Wem? → meistens eine Person
Artikel: dem / einem (m/n)
der / einer (f)

Ich gebe dem Kind ...
Akkusativobjekt
Was? → meistens eine Sache
Artikel: den / einen (m)
das / ein (n) · die / eine (f)

... das Buch.
Satzmuster
Verb → Dativ (Person) + Akkusativ (Sache)
geben
→
Wem? Was?
Ich gebe meiner Mutter ein Geschenk.
zeigen
→
Wem? Was?
Er zeigt dem Arzt den Befund.
schicken
→
Wem? Was?
Sie schickt ihrer Freundin eine Nachricht.
erklären
→
Wem? Was?
Er erklärt den Schülern die Regel.
Reihenfolge im Satz:
Normalerweise: Dativ vor Akkusativ — Ich gebe ihr das Buch.
Mit Pronomen: Beide Pronomen → Akkusativ zuerst — Ich gebe es ihr.
