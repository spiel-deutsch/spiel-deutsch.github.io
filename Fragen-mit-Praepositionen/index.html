Fragen mit Präpositionen

Wie fragt man nach Personen und Sachen bei Verben mit fester Präposition? wo(r)+Präp. oder Präp.+wen/wem? — komplett erklärt mit arabischen Bedeutungen.

Grundregel Das System wofür worauf worüber womit wovon wovor worum Alle Fragen Beispiele Tipps
Die Grundregel — القاعدة الأساسية
Person oder Sache? — شخص أم شيء؟
Wenn ein Verb eine feste Präposition hat (z.B. warten auf), gibt es zwei verschiedene Wege zu fragen — abhängig davon, ob das Objekt eine Person oder eine Sache/Idee ist.

Sache/Idee → Fragewort: wo(r) + Präposition = worauf? wofür? womit?
Person → normal: Präposition + wen / wem? = auf wen? für wen? mit wem?

wo- oder wor-? — Die Regel
auf / über / aus / an → Vokal → wor + auf = worauf Worauf wartest du?
mit / für / von / vor / zu / um → Konsonant → wo + mit = womit Womit fährst du?
Antwort auf Sachen → da(r) + Präposition
Worauf wartest du? → Darauf. (auf den Bus)

Lerntipps
👤💡
Person oder Sache?
Person → Präp. + wen/wem
Auf wen wartest du? — Auf ihn.

Sache → wo(r) + Präp.
Worauf wartest du? — Auf den Bus.
🔤
wor- oder wo-?
Vokal (auf/über/aus/an) → wor+: worauf, worüber

Konsonant (mit/für/von) → wo+: womit, wofür
🔄
Antwort: da(r)- + Präp.
Worauf wartest du? → Darauf.
Wofür dankt er? → Dafür.
Womit fährst du? → Damit.

Gleiche Regel: Vokal → dar-, Konsonant → da-
📝
Immer im Paar lernen
Lerne die Frage immer in beiden Versionen:

warten auf
